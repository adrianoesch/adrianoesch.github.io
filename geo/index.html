Additional Layers for geo.admin

I've scraped some websites for their valuable information, put the data in a kml file, which then can be imported into the geo.admin page

Some of the links are outdated since swisstopo has released official data, the ski route layer for example. Nonetheless, there's more functionality when importing via kml.

Use with caution, nothing guaranteed, call in advance (sources are linked)!
